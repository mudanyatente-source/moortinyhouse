# Galeri klasörleri

Bu alt klasörlere (blog, models, portfolio, referanslar) `.jpg`, `.png`, `.webp` eklediğinizde
Galeri sekmesinde klasör adına göre kategori olarak görünür.

- `blog/` — Blog yazıları
- `models/` — Model görselleri  
- `portfolio/` — Portföy projeleri
- `referanslar/` — Müşteri / referans fotoğrafları

`public` kökündeki görseller "Genel" kategorisinde listelenir.
